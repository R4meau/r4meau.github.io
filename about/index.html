Born and raised in Haiti, I’m a self taught software engineer with a passion for learning. I started programming in 2008 and I haven’t stopped since then. But it’s really only in 2016, after dropping out of college, that I started improving myself seriously and focusing on my goals. This is when I became part of the inaugural class of 42 US.

I’ve been working as a freelance web developer for years. But I’m not limited to the web, I enjoy building mobile and desktop applications, coding beautiful visual arts, and participating in programming contests.

I don’t like to say that I know this or that particular language because to me, languages are simply tools. I focus on projects as a whole and I’m ready to use all the necessary tools to achieve my goal. Of course, I do know the following languages very well from using them so much:

  • HTML, CSS and JavaScript
  • C and C++
  • Processing
  • Python

Like I said, I have a passion for learning and because of the experience I gathered from all those years of programming, I can learn really fast.

This website is here to document my learning, show what I’ve built and what I’m working on.

You can download my resume here: Nick Rameau’s Resume in English

Feel free to contact me if you see a potential in me, I appreciate it :)
